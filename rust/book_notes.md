# Rust Book Notes
-----------------------------------------------------------
|------------: Chapter 1: -------------------|

Rust is a low-level language that uses concepts to be easy to follow and understand.

Functions are coded like this:

``` rust
fn main() {
    println!("Hello, world!");
}
```
[^1] How to Run normal Rust code:

Run on the terminal with this code:

``` bash
rustc main.exe
./main
```
